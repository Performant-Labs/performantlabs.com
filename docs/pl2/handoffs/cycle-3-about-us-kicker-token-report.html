What changed — plain English

  • The terracotta token used for kickers on warm cream backgrounds moved from #8C4E33 (rgb 140, 78, 51) to the brief-canonical #8E4A2A (rgb 142, 74, 42). About a 2-unit shift on each RGB channel.
  • On /about-us §B "Track record" and §D "Dogfood": kicker color is now the same on-brief terracotta seen on §A and §C. Cycle 1's DELTA rows for these two kickers flip to MATCH.
  • Because the fix is at the :root token level, the same shift propagates to three sibling pages where a theme--light kicker exists: homepage "Dogfooding", /services "Capacity", and /open-source-projects "Testing tools". Diff PNGs confirm the shift is kicker-only — no spillover into body copy, cards, hairlines, or chrome.
  • WCAG contrast on cream improves marginally from 5.63:1 to 5.79:1, well above the 4.5:1 AA body-text threshold.
  • §A, §C, §E kickers are unchanged. Espresso §E "Get started" still uses --accent #C97B5C.
  • No orphan words. No new preview defects. No regression elsewhere on any page.
