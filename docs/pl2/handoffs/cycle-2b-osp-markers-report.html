What I see different

No visible differences detected at any viewport on any page.

  • /open-source-projects looks identical pre- and post-refactor at 1280, 768, and 375 — including the Hero and Community sections, which got the new marker class.
  • /services, /about-us, /how-we-do-it, and the homepage look identical pre- and post-refactor at all three viewports — confirming the retained :has() half still centers the four cross-page consumers.
  • No layout shifts, no spacing changes, no color drifts, no typography drifts, no border or shadow changes anywhere.

Mechanism: the marker class adds a parallel selector that matches the same DOM with byte-identical declarations. Specificity went from 0,1,0 to 0,2,0 with no intermediate-specificity rule in the cascade, so computed styles are unchanged.
